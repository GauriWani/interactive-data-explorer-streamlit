Self-Service Interactive Data Explorer

This Streamlit application enables users to upload CSV files and perform interactive data exploration without writing any code. It is designed to support quick analysis and visualization for business and analytics teams.

Features

CSV file upload and data preview

Dynamic filtering for categorical and numerical columns

Interactive visualizations using Plotly

Real-time KPI metrics (count, mean, max)

Self-service analytics for non-technical users

Tech Stack

Python, Streamlit, Pandas, Plotly
